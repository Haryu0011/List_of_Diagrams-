### 🛠️ Tools Yang Dipake Buat Bikin Diagram

Repo ini digambar pake tools ini:

- 🌱 **PlantUML** - Nulis diagram pake kode text
- 📈 **Graphviz** - Engine buat ngerender graph/tree 
- 💻 **VSCode** - Editor tempat nulis kodenya
- 🧩 **Extension PlantUML** - Biar bisa preview diagram langsung di VSCode

### 👀 Cara Liat Diagramnya - 3 Klik Doang

**Buat yang belum pernah buka GitHub sama sekali:**

1. **Klik link repo di atas** 
   Langsung kebuka halamannya. Gak usah login, gak usah daftar.

2. **Klik folder `diagram`** 

3. **Klik src/main/java** 
   Ya, itu satu nama doang.
   
4. **Klik lagi Diagrams**
   Nanti disana kamu bisa pilih mau buka apa, Activity, Usecase, atau Sequence.

5. **Didalam folder activity atau usecase atau sequence**
   Klik namaFileNya.jpg buat lihat gambar diagram langsung
   Klik namaFileNya.puml buat lihat kode sumber plantUML
   
   Selesai. Gampang kan?
